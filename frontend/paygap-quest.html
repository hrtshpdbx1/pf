À propos de cette démo

Cette version est une démonstration interactive Frontend uniquement. Elle présente le design visuel et les interactions conçus durant l'événement. Pour faciliter l'accès immédiat, cette démo présente la structure fidèle au prototype original, bien que les calculs en temps réel (Backend via PHP) soient ici simulés. L'architecture complète reste consultable sur GitHub.

Objectif du projet

J'ai proposé ce projet pour répondre au Hackathon @Finances : « 3 jours pour prouver que la finance peut être créative, accessible et engagée. »

En Belgique, l’égalité salariale est un enjeu de justice sociale : les femmes gagnent encore en moyenne 20 % de moins que les hommes, travaillant symboliquement "gratuitement" à partir du 20 octobre. L'absence de négociation peut coûter jusqu'à 1 million d'euros sur une carrière. Face à ce manque d'éducation financière, notre équipe a créé un outil pour se réapproprier ces enjeux par l'information et la négociation.

Fonctionnalités clés :

  • Calculateur salarial : Comparatif dynamique basé sur le secteur, le niveau et le genre.
  • Feuille de route : Guide de négociation structuré et téléchargeable en PDF.
  • Boîte à outils : Annuaire de ressources (livres, podcasts, lois) pour s'informer.
  • API Sectorielle : Structuration des données réelles de l'IEFH 2024.

Le compteur dynamique : Visualiser l'urgence

Comment capter l'attention dès l'accueil ? J'ai développé un compteur JavaScript qui traduit l'inégalité en temps réel. En voyant les minutes défiler, l'utilisateur prend conscience que l'inégalité est une perte qui s'accumule chaque seconde.

Calcul dynamique de l'inégalité en temps réel function updateCounter() {
  const today = new Date();
  let diff = today - unequalPayDay;
  // Conversion en jours, heures, minutes
  const diffDays = Math.floor(diff / (1000 * 60 * 60 * 24));
  counterElement.innerHTML = `Depuis ${diffDays} jours...`;
}
